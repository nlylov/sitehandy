Common Questions

Range & Oven FAQ

How much does installation cost?

Flat-rate based on fuel type and connection requirements. We quote before starting.

Do you connect gas ranges?

Yes. We use code-compliant flexible connectors and perform a leak test before leaving.

Do you install the anti-tip bracket?

Yes, per code requirements. We secure it to the floor or wall depending on the model.

Can you convert gas to propane?

Yes, if the manufacturer supplies a conversion kit. We swap orifices and test all burners.
