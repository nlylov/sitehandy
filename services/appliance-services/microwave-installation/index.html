Common Questions

Microwave FAQ

How much does installation cost?

Flat-rate pricing based on type. OTR requires mounting and venting; built-in kits are simpler. We quote upfront.

Do you remove the old microwave?

Yes. We safely disconnect, unmount, and remove the existing unit.

Can you install OTR where there wasn't one?

Yes. We install mounting bracket, connect venting, and run a dedicated outlet if needed.

How long does it take?

OTR: 1–2 hours including removal. Built-in and drawer: about 1 hour.
