Common Questions

Through-Wall AC FAQ

How much does through-wall AC installation cost in NYC?

Pricing depends on the unit size (BTU), sleeve condition, and whether electrical work is needed. Contact us for a quick flat-rate quote — we confirm the price before any work begins.

What is a through-wall AC unit?

A through-wall AC sits in a metal sleeve built into the wall. Unlike window units, they don't block the window and provide a cleaner look. They're common in NYC co-ops and condos.

Can you install a through-wall AC in an existing sleeve?

Yes. We install new units into existing wall sleeves, ensure proper fit and seal, and test the unit for cooling performance.

Do you handle the electrical for through-wall AC?

We can install dedicated outlets for AC units requiring 220V/240V power. For buildings requiring permits, we coordinate with your building management.
