Common Questions

PTAC FAQ

How much does PTAC installation cost?

Pricing depends on unit size, sleeve condition, and electrical requirements. We inspect everything before quoting a flat rate.

Do you inspect the wall sleeve?

Yes. We check for rust, gaps, insulation condition, and drainage slope. Sleeve repair is available if needed.

Can you replace with a different brand?

Usually yes. PTAC units follow standard sleeve sizes. We verify compatibility and install adapter kits if needed.

Do you handle the electrical hookup?

Yes. We connect to the existing circuit, verify amperage/voltage, and test all modes before leaving.
