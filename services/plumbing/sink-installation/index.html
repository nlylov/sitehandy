Need a sink installed in New York City? Repair Asap LLC installs kitchen sinks, bathroom sinks, utility sinks, and vessel sinks across Manhattan, Brooklyn, Queens, and Staten Island.

Every sink installation includes supply line connection, drain assembly, silicone sealing, and a leak test. We handle drop-in, undermount, vessel, and wall-mount sinks.

Common Questions

Sink Installation FAQ

How much does sink installation cost in NYC?

Standard sink installation runs $150–$300 depending on sink type and plumbing modifications.

Do you install undermount sinks?

Yes. We install undermount sinks with proper clips, silicone adhesive, and support.

Can you install a new sink where there isn't one?

We can install sinks in locations with existing plumbing.

Do you remove the old sink?

Yes. We disconnect, remove, and dispose of your old sink as part of the installation.
