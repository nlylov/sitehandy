Common Questions

Light Fixture FAQ

How much does installation cost?

Flat-rate per fixture. Simple swaps are most affordable. New junction boxes and recessed installs priced by complexity.

Can you add a fixture where there isn't one?

Yes. We run new wiring, install a junction box, and mount your fixture.

Do you install heavy chandeliers?

Yes. We install fan-rated brace boxes and verify weight support before hanging.

Can you add recessed lighting?

Yes. Retrofit cans and LED wafer-style fixtures that need no housing.
