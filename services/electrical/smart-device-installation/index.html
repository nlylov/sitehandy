Common Questions

Smart Home FAQ

What devices do you install?

Smart thermostats, locks, lighting, doorbells, plugs, and switches. All major brands.

Do I need a C-wire for a smart thermostat?

Most require one. We can run it or install a C-wire adapter if yours is missing.

Do you set up the app and WiFi?

Yes. Full setup — hardware, WiFi pairing, app config, and testing before we leave.

Do you install smart locks?

Yes. Deadbolt replacements, keypad locks, app pairing and pin code setup included.
