Common Questions

Ceiling Fan FAQ

How much does installation cost?

Flat-rate. Replacing an existing fan is most affordable. New installs priced by complexity.

Do I need a fan-rated box?

Yes. We install one if your current box isn't rated for ceiling fan weight and vibration.

Can you install where there's only a light?

Yes. We swap the box for a fan-rated brace and connect wiring.

Do you install smart/remote fans?

Yes — included remotes, WiFi-enabled fans, and aftermarket remote kits.
